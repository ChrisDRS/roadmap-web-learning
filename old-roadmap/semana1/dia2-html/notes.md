## Resumen teórico clave:

### 1. ¿Cómo funciona Internet?
- Internet es una red global de computadoras conectadas mediante protocolos estándar (TCP/IP).
- La información viaja en paquetes, que son unidades pequeñas de datos que contienen la información que se está enviando, más la dirección del remitente y del destinatario.
- Para conectarse a Internet, los dispositivos usan una dirección IP, que identifica de manera única cada computadora o servidor conectado a la red.

### 2. HTTP (Protocolo de Transferencia de Hipertexto)
- HTTP es el protocolo principal utilizado para transmitir datos en la web.
- Opera mediante solicitudes (`requests`) y respuestas (`responses`) entre cliente y servidor.
- Métodos comunes son `GET`, `POST`, `PUT`, `DELETE`.

### 3. Navegadores web
- Un navegador interpreta código HTML, CSS y JavaScript, convirtiendo estos lenguajes en páginas web visualmente atractivas y funcionales.
- Ejemplos comunes son Chrome, Firefox, Edge y Safari.
- Cada navegador utiliza un motor de renderizado específico (Blink, Gecko, WebKit).

### 4. DNS (Sistema de Nombres de Dominio)
- DNS traduce nombres de dominio (www.ejemplo.com) a direcciones IP (192.0.2.1).
- El proceso ocurre mediante consultas a servidores DNS especializados.
- DNS es esencial para la navegación amigable en Internet.
